# AI & Machine Learning: Theory & Practice

This repository contains a comprehensive collection of implementations from my AI course labs and home assignments. It serves as a practical archive of various algorithms, ranging from classical search strategies to modern machine learning workflows.

---

# 📂 Repository Structure

The repository is organized by topic to reflect the academic progression of the course.

## 1. Intelligent Agents

Implementations of different agent architectures based on their interaction with environments:

- **Simple Reflex Agents**  
  Decision-making based only on current percepts.

- **Model-Based Agents**  
  Agents that maintain internal states to work in partially observable environments.

- **Goal & Utility-Based Agents**  
  Systems designed to achieve specific goals or maximize utility values.

---

## 2. Search Algorithms

A collection of classical and advanced pathfinding and optimization techniques.

### Uninformed Search
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Uniform Cost Search (UCS)

### Informed Search
- A* Search
- Greedy Best-First Search

### Adversarial Search
- Minimax Algorithm
- Alpha-Beta Pruning

---

## 3. Machine Learning

Practical implementations of machine learning workflows and predictive modeling techniques.

### Data Preprocessing
- Feature Scaling
- Data Cleaning
- Encoding using Pandas and NumPy

### Supervised Learning
- Linear Regression
- Logistic Regression
- Decision Trees
- Classification Tasks

### Unsupervised Learning
- Clustering Techniques
- Pattern Recognition

### Model Evaluation
- Precision
- Recall
- F1-Score
- Confusion Matrix
